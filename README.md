#  Nmap Recon Toolkit

This project is a simple cybersecurity reconnaissance toolkit using **Nmap**.

It documents real scanning techniques used in penetration testing and network security assessments.

---

What I Learned
- Network discovery using Nmap
- Port scanning techniques
- Service detection
- OS fingerprinting basics

---

 Tools Used
- Nmap
- Linux terminal (Kali Linux / Windows CMD)

---

 Sample Commands
 Basic scan
```bash
nmap 192.168.191.130

Detailed scan
nmap -sV -sC 192.168.191.130
Full scan
nmap -p- 192.168.191.130
